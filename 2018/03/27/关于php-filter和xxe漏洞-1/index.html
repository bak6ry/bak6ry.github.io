#关于php://filter协议
php://filter协议是php语言中的一个特有的协议流，它的作用是作为一个中间流来处理其他流。比如我们可以用一行代码将post请求内容转换成base64编码输出
readfile("php://filter/read=convert.base64-encode/resource=php://input");
